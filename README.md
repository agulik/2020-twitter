## Instructions
&nbsp;
### Clone the repository
```
git clone
```

### Install dependencies
```
yarn
```

### Set Twitter ENV variable on server
```
export TWITTER_BEARER_TOKEN="MY_TOKEN"
```

### Start the server
```
yarn server
```

### Start the client
```
yarn start
```
